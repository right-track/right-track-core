GTFS Trip

Representation of the GTS Trip definition

GTFS Required Fields:

  • Trip ID
  • Route
  • Service
  • List of StopTimes

GTFS Optional Fields:

  • Trip Short Name
  • Trip Direction ID
  • Trip Wheelchair Accessibility

Right Track Fields:

  • Trip Direction Description
  • Trip Peak Indicator
