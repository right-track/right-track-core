Description

GTFS Trip

Representation of the GTFS Trip definition

GTFS Required Fields:

  • Trip ID
  • Route
  • Service
  • List of StopTimes

GTFS Optional Fields:

  • Trip Headsign
  • Trip Short Name
  • Trip Direction ID
  • Trip Block ID
  • Trip Shape ID
  • Trip Wheelchair Accessibility
  • Trip Bikes Allowed

Right Track Fields:

  • Trip Direction Description
  • Trip Peak Indicator
