Description

GTFS Agency

Representation of the GTFS Agency definition.

GTFS Required Fields:

  • Agency Name
  • Agency URL
  • Agency Timezone

GTFS Optional Fields:

  • Agency ID
  • Agency Language Code
  • Agency Phone Number
  • Agency Fare URL
  • Agency Email
