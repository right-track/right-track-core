RT Favorite

Representation of a Right Track User Favorite. This can be a saved Station or Trip with the associated parameters and optional options.

The following describes the format for the Favorite parameters and options.

Station

Parameters:

 {
   stop: {
     id: 'Stop ID',
     statusId: 'Stop Status ID',
     name: 'Stop Name'
   }
 }

Options:

Currently, there are no options for a Favorite Station.

Trip

Parameters:

 {
   origin: {
     id: 'Stop ID'
     statusId: 'Stop Status ID',
     name: 'Stop Name'
   },
   destination: {
     id 'Stop ID'
     statusId: 'Stop Status ID',
     name: 'Stop Name'
   }
 }

Options:

 {
   allowTransfers: true
 }
